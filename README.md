This is a minimal project bundle. Steps to run locally:

1) Install Node.js (LTS)
2) In project folder:
   npm install
   npm run dev
3) Open the displayed localhost URL (usually http://localhost:5173)

To deploy on Vercel:
- Create a GitHub repo, push these files
- Login to vercel.com and import the repo (default build: `npm run build`)

Notes:
- Data is stored in browser localStorage.
- UI is dark-mode, font sizes increased for mobile readability.
